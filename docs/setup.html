üîß Detailed Configuration

1. Anti-Raid Configuration

Configure advanced anti-raid protection:

/antiraid config

Settings:

  • Detection Sensitivity: Low, Medium, High, Paranoid
  • Action on Detection: Ban, Kick, Quarantine
  • Join Rate Threshold: Max joins per time window
  • Whitelist: Exempt trusted users

2. Anti-Nuke Configuration

Set up server protection against nuking:

/security config

Protection for:

  • Channel deletion/creation
  • Role deletion/modification
  • Mass bans/kicks
  • Webhook spam
  • Permission abuse

3. Join Gate Setup

Configure member verification:

/joingate setup

Verification Types:

  • Button Verify: Click a button to verify
  • Reaction Verify: React to verify
  • Captcha: Complete captcha challenge
  • 2FA: Two-factor authentication

4. Logging Configuration

Set up comprehensive server logging:

/logs setup

Log Types:

  • Moderation logs (bans, kicks, warns)
  • Server logs (channel/role changes)
  • Member logs (joins, leaves, updates)
  • Message logs (edits, deletions)
  • Security logs (threats, raids, nukes)

5. Notifications Setup

Configure real-time alerts:

/notify setup

Alert Types:

  • Security threats
  • Raid attempts
  • Nuke attempts
  • High-priority moderation

üîç Troubleshooting

Bot Not Responding?

  • Check bot has proper permissions (Administrator recommended)
  • Ensure bot role is above roles it needs to manage
  • Run /ping to check if bot is online
  • Use /troubleshoot for automated diagnostics

Commands Not Working?

  • Commands sync automatically - wait 1 minute after invite
  • Try kicking and re-inviting the bot
  • Check you have required permissions for the command
  • Join support server for help

False Positives?

  • Lower anti-raid sensitivity in /antiraid config
  • Whitelist trusted users with /security whitelist
  • Adjust thresholds in configuration
