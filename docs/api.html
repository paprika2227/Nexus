Getting Started

The Nexus Public API allows you to access bot data, server statistics, and user information programmatically. All endpoints require an API key for authentication.

Authentication

Include your API key in requests using either the X-API-Key header or api_key query parameter:

curl -H "X-API-Key: nx_your_api_key_here" https://regular-puma-clearly.ngrok-free.app/api/v1/server/123456789

Rate Limiting

API keys are limited to 100 requests per day. Rate limit information is included in response headers:

  • X-RateLimit-Limit: Your daily limit
  • X-RateLimit-Remaining: Requests remaining today

Request an API Key

To request an API key, contact us at nexusbot0@proton.me with:

  • Your Discord username
  • Intended use case
  • Estimated daily requests

Endpoints

GET /api/v1/server/:id

Get server information and configuration

Parameters

Name Type Description
id string Discord server ID (path parameter)

Example Request

curl -H "X-API-Key: nx_your_key" https://regular-puma-clearly.ngrok-free.app/api/v1/server/1234567890

Example Response

{ "id": "1234567890", "name": "My Server", "memberCount": 1500, "features": { "antiNuke": true, "antiRaid": true, "autoMod": false }, "stats": { "totalBans": 45, "totalKicks": 123, "warnings": 78 } }

GET /api/v1/user/:userId/warnings

Get user warnings in a specific server

Parameters

Name Type Description
userId string Discord user ID (path parameter)
guild_id string Discord server ID (query parameter, required)

Example Request

curl -H "X-API-Key: nx_your_key" https://regular-puma-clearly.ngrok-free.app/api/v1/user/9876543210/warnings?guild_id=1234567890

Example Response

{ "warnings": [ { "id": 1, "reason": "Spam", "moderator_id": "1111111111", "timestamp": 1704067200000 } ] }

GET /api/v1/votes/leaderboard

Get voting leaderboard

Query Parameters

Name Type Description Default
type string total, streak, or longest total
limit number Number of results (max: 100) 10

Example Request

curl -H "X-API-Key: nx_your_key" https://regular-puma-clearly.ngrok-free.app/api/v1/votes/leaderboard?type=streak&limit=5

Example Response

{ "leaderboard": [ { "user_id": "1234567890", "total_votes": 150, "current_streak": 30, "longest_streak": 45 } ] }

Error Codes

Code Description
401 Invalid or missing API key
429 Rate limit exceeded
404 Resource not found
500 Internal server error
