1. Introduction

Nexus Bot ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard information when you use our Discord bot.

2. Information We Collect

2.1 Server Information

When you add Nexus to your Discord server, we collect:

  • Server ID and name
  • Channel IDs and names
  • Role IDs and names
  • Member counts (not individual user data)
  • Server configuration settings you choose

2.2 Security Event Data

To provide protection services, we log:

  • Detected threats and raid attempts
  • Moderation actions taken by the bot
  • User IDs involved in security events (for anti-nuke/anti-raid only)
  • Timestamps of security events

2.3 Command Usage

We collect anonymous usage statistics:

  • Command names executed (not content)
  • Execution timestamps
  • Success/failure status (for debugging)

2.4 What We DON'T Collect

  • ‚ùå Message content (unless explicitly reported via commands)
  • ‚ùå Private conversations or DMs
  • ‚ùå Personal user information beyond Discord IDs
  • ‚ùå Payment information (we're free!)
  • ‚ùå Email addresses or contact details

3. How We Use Your Information

We use collected data solely to:

  • Provide anti-raid and anti-nuke protection
  • Generate server analytics and insights
  • Create recovery snapshots for auto-restoration
  • Improve bot performance and features
  • Debug issues and provide support

4. Data Storage and Security

All data is stored locally in encrypted SQLite databases on our secure servers. We implement industry-standard security measures including:

  • Encrypted storage at rest
  • Secure server access controls
  • Regular security audits
  • Automatic data backups

5. Data Retention

  • Active Servers: Data retained while bot is in server
  • Removed Servers: Data deleted within 30 days of bot removal
  • Security Logs: Retained for 90 days for threat analysis
  • Snapshots: Automatically pruned based on your retention settings

6. Data Sharing

We do NOT sell, rent, or share your data with third parties. The only exceptions are:

  • When required by law or valid legal process
  • To protect against fraud or security threats
  • Anonymous, aggregated statistics (e.g., "Total raids blocked: 1000")

7. Your Rights

You have the right to:

  • Access: Request a copy of your server's data
  • Deletion: Request deletion of your data (remove bot = auto-delete)
  • Correction: Update incorrect information via bot commands
  • Opt-out: Disable specific features (analytics, logging, etc.)

8. Third-Party Services

Nexus integrates with:

  • Discord API: Subject to Discord's Privacy Policy
  • Bot Lists: Top.gg, Discord Bot List, etc. (only server counts, no personal data)

9. Children's Privacy

Nexus is intended for Discord servers and does not knowingly collect information from children under 13. Discord's Terms of Service prohibit users under 13.

10. Open Source Transparency

Nexus is open source. You can review our code on GitHub to verify our data handling practices.

11. Changes to This Policy

We may update this Privacy Policy occasionally. Changes will be posted on this page with an updated "Last updated" date. Continued use of the bot after changes constitutes acceptance.

12. Contact Us

If you have questions about this Privacy Policy or want to exercise your rights:

  • Discord: Support Server
  • GitHub: Open an Issue
