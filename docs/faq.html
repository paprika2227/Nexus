üîí Security & Privacy

Is Nexus safe? Can I trust it?

Nexus is 100% open source on GitHub. You can read every line of code and verify exactly what it does. Unlike closed-source bots, there's nothing hidden. Security researchers can audit the code, and you can self-host if you want complete control.

What data does Nexus collect?

Nexus only collects data necessary for features to work:

  • Server configuration (your settings)
  • Moderation logs (cases, warnings)
  • Security events (threats detected)
  • User behavior (for AI analysis, if enabled)

We don't sell data, share with third parties, or use it for advertising. Full privacy policy available on GitHub.

What happens to my data if I remove Nexus?

All data is deleted within 30 days of bot removal. You can also request immediate deletion using /privacy export before removing the bot.

‚öôÔ∏è Setup & Usage

How long does setup take?

2 minutes! Invite the bot, run /setup preset, and you're done. Nexus auto-configures based on your server type.

Do I need technical knowledge?

Nope! Nexus is designed for everyone. The /setup preset command handles everything automatically. Advanced users can dive into detailed configuration, but it's not required.

What permissions does Nexus need?

Administrator permission is recommended for full functionality. Minimum permissions: Manage Server, Manage Roles, Manage Channels, Ban Members, Kick Members, Manage Messages, Read Message History.

Can Nexus handle large servers?

Yes! Nexus uses automatic sharding and is cluster-ready. It scales from small communities to servers with 100,000+ members.

ü§ñ AI Features

How does the AI work?

Nexus uses pattern recognition and behavioral analysis to:

  • Detect suspicious user behavior
  • Predict threats before they happen
  • Provide security recommendations
  • Prioritize moderation actions

The AI learns from patterns across all Nexus servers (anonymized) to improve detection accuracy.

Does the AI make false positives?

Nexus has a whitelist system to prevent false positives. Trusted users are exempt from AI analysis. The AI also uses confidence scoring - it only takes action at 60%+ confidence.
