# Privacy Policy

**Last Updated: [Date]**

## Introduction

Nexus Bot ("we", "our", "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect information when you use Nexus Bot on Discord.

## Information We Collect

### Server Information

- Server ID
- Server name
- Server settings and configurations
- Moderation logs and actions
- Security events and threat data

### User Information

- User IDs (Discord snowflakes)
- Usernames and discriminators
- Messages (for moderation purposes only)
- Moderation history
- Behavioral patterns (for threat detection)

### Technical Information

- Command usage statistics
- Bot performance metrics
- Error logs
- API usage data

## How We Use Information

### Security & Moderation

- Detect and prevent raids, nukes, and other attacks
- Moderate server content
- Track user behavior for threat detection
- Share threat intelligence across servers (anonymized)

### Service Improvement

- Analyze bot performance
- Fix bugs and errors
- Improve features
- Generate statistics and reports

### AI Features

- Train machine learning models (server-specific)
- Generate security recommendations
- Analyze behavioral patterns
- Predict potential threats

## Data Storage

### Location

- All data is stored locally on our servers
- Database is encrypted at rest
- No third-party data sharing

### Retention

- Server configurations: Until server removes bot
- Moderation logs: 90 days (configurable)
- Analytics: 1 year
- Threat intelligence: 30 days (anonymized)

## Data Sharing

### We Do NOT:

- Sell your data to third parties
- Share personal information
- Use data for advertising
- Share data with other Discord bots (except anonymized threat intelligence)

### We DO:

- Share anonymized threat intelligence across servers (for network protection)
- Use data for bot functionality and security
- Store data securely on our servers

## Your Rights

### Access

- Request a copy of your server's data
- View what data we have stored

### Deletion

- Request deletion of your server's data
- Remove bot from server (data deleted after 30 days)

### Control

- Configure what data is collected
- Opt-out of threat intelligence sharing
- Disable analytics

## Security

### Measures

- Encrypted database storage
- Secure server infrastructure
- Regular security audits
- Access controls and authentication

### Breach Notification

- We will notify affected servers within 72 hours of discovering a breach
- Notification via Discord or email
